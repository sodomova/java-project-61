### The Brain Games
[![Actions Status](https://github.com/sodomova/java-project-61/workflows/hexlet-check/badge.svg)](https://github.com/sodomova/java-project-61/actions)
<a href="https://codeclimate.com/github/sodomova/java-project-61/maintainability"><img src="https://api.codeclimate.com/v1/badges/6e059baa531e12d2a18b/maintainability" /></a>

This is a set of five console games. Each game asks questions that must be answered correctly. After three correct answers, the game is considered completed. Incorrect answers end the game and prompt you to play it again.

This is my first training project. The project does not use abstract classes, interfaces, inheritance, or class instances.
